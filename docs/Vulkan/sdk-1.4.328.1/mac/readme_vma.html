Product page: Vulkan Memory Allocator on GPUOpen

Note

Vulkan Memory Allocator comes in form of a C++ stb-style header file. As a result, the Vulkan SDK only ships vk_mem_alloc.h.

See the official Vulkan Memory Allocator documentation for further information.
